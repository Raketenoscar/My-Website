Syntax und Semantik von HTML

Syntax:

  • HTML besteht aus Elementen mit öffnendem und schließendem Tag

    <p>Inhalt</p>

  • Fast alle Tags werden mit einem "closing tag" geschlossen

    </p>

  • Es gibt Attribute welche im offenen Tag eines Elements stehen und immer von Anführungszeichen umgeben sind

    <img src="bild.jpg">

  • HTML tags können groß oder klein geschrieben sein
  • Es gibt Elemente die keinen "closing tag" haben

    <img>

  • Es gibt eine bestimmte Ordnung wie eine HTML datei aufgebaut ist. 1. <html> 2. <head> 3. <body>
  • Falsche Tags oder Attribute werden nicht angezeigt oder Falsch angezeigt
  • Es gibt Kommentare welche dafür da sind deinen Code übersichtlicher zu machen und werden von der Website ignoriert

    <!--Kommentar-->

  • Der HTML-Code startet mit

    <!DOCTYPE html>

Semantik:

  • <html>

    Wurzelelements eines HTML-Dokuments

  • <head>

    Enthält Metadaten und Informationen über das Dokument

  • <title>

    Definiert den Titel der Webseite, der im Browser-Tab angezeigt wird

  • <body>

    Enthält den sichtbaren Inhalt der Website

  • <h1>

    bis

    <h6>

    Überschriften mit verschiedenen größen

  • <p>

    Erstellt ein Textfeld

  • <a>

    Erstellt einen klickbaren Link

  • <img>

    Ermöglicht ein Bild einzufügen

  • <div>

    Container welcher sogut wie alle Elemente halten kann

  • <ul>

    Erzeugt eine ungeordnete Liste

  • <ol>

    Erzeugt eine geordnete Liste

  • <li>

    Wird in

    <ol>

    oder

    <ul>

    geschrieben um die Liste anzuzeigen
